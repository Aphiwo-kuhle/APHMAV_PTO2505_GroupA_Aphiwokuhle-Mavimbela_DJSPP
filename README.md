## 📌 Project Overview

This is a Podcast Web Application built with React and React Router.

Users can:

* Browse podcast shows
* View show details
* Play audio globally
* Add/remove favourites
* Sort and group favourites
* Toggle between light and dark mode

---

## 🚀 Features

### 🎵 Audio Player

* Global audio player
* Play, pause, seek
* Continues across pages

### ❤️ Favourites

* Add/remove episodes
* Stored in localStorage
* Grouped by show
* Sorted (A-Z, date)

### 🌙 Theme

* Light/Dark mode
* Saved in localStorage

### 📱 Responsive Design

* Works on mobile and desktop
